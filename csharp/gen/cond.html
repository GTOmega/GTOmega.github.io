IF

The starting of the IF statement is quite simeple, you start out with just that, if then (), in these you put your actual condition e.g. int>=10, this works with other variables and you can just && to look for multiple conditions or || for an either or type beat.
and finally there are {} where you put what you want to happen

e.g.
int age = 15;
if (age < 18)
{
Console.WriteLine("Stupid Child");
}

Else

This is technically a subset of the IF Fucntion, if the value in conditions is wrong you can run something else off of that using the same principal

e.g.
int age = 15;
if (age < 18)
{
Console.WriteLine("Stupid Child");
}
else (age < 18{
Console.WriteLine("Chad Adult");
}
