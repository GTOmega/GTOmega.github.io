Font


time to do the funky fonts

Font Size

font-size:9px; - used to reset the size of fonts so there huge or just really small, your choice

if you really want to do it the gay way then do this style="font-size:10px;"


Font type

font-family:fontname; - used to change stuff from comic sans, like im using right now, to arial or other garbage fonts

if you really want to do it the gay way then do this style="font-family:fontname;"

Font Weight

font-weight:size; - used to change the weight from as think as a paper to as thicc as a brick house

if you really want to do it the gay way then do this style="font-weight:size;"

Font Style

font-style:type; - ya know, im not really sure, but it can also do italics

if you really want to do it the gay way then do this style="font-style:type;"
